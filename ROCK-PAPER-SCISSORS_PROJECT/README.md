# Rock Paper Scissors – GUI Version  
### Codesoft Python Internship Project

This is a GUI-based Rock–Paper–Scissors game built using Python and Tkinter.

## Features
- Choose Rock, Paper, or Scissors
- Computer randomly selects its choice
- Shows who wins each round
- Live score tracking
- Simple and user-friendly interface
